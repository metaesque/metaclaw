# Web Fetcher Overview

## Providers

### Crawl4AI
Crawl4AI is an open-source, ultra-fast web crawler designed specifically for
LLMs and AI agents, returning clean Markdown and structured extraction natively.

### Firecrawl
Firecrawl is an API service and open-source tool designed to search, scrape, and
interact with the web, turning any URL into clean Markdown or structured JSON
for LLMs.

### Browse AI
Browse AI is a no-code web scraping and monitoring platform that allows users to
train a cloud robot to extract data or monitor site changes in minutes.

### Jina AI Reader
Jina AI Reader is an ultra-lightweight API tool that instantly converts any URL
into LLM-friendly Markdown by simply prepending a specific prefix to the target
address.

### Zyte
Zyte is an enterprise-grade, all-in-one Web Scraping API that specializes in
bypassing the most aggressive anti-bot protections on the internet.

### Octoparse
Octoparse is a powerful no-code web scraping platform that excels at large-
scale, structured data extraction across paginated and highly dynamic websites.

### ScrapeGraphAI
ScrapeGraphAI is a Python library and API that utilizes LLMs and direct graph
logic to dynamically construct scraping pipelines from natural language prompts.
