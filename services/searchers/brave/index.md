# brave: Brave Search API

## Overview

The vast majority of alternative search engines simply lease their search
results from Google or Bing. Brave has built its own independent index of the
web from scratch. It is designed to prioritize high-quality, human-created
content while aggressively filtering out SEO spam and affiliate marketing farms.

For OpenClaw, the Brave API offers a massive, high-quality data stream that is
free from algorithmic bias or corporate filtering. It provides a clean, well-
structured JSON response that agents can parse instantly without needing
secondary extraction tools.

Because it emphasizes privacy and data independence, it aligns perfectly with
the MetaClaw ethos. It is highly recommended for users who want enterprise-grade
search speed but refuse to feed their AI's queries into the data-harvesting
ecosystems of major tech conglomerates.
