# Web Search API Overview

## Providers

### Exa
Exa (formerly Metaphor) is a neural search engine that finds web pages based on
semantic meaning and context rather than relying on exact keyword matches.

### SearXNG
SearXNG is a free, privacy-respecting metasearch engine that aggregates results
from more than 70 search services while preventing user tracking and profiling.

### Brave Search API
The Brave Search API provides access to a completely independent, global web
index that does not rely on Big Tech monopolies, ensuring unbiased and private
results.

### Linkup
Linkup is an enterprise-grade AI search platform focused on connecting agents to
premium, factual, and up-to-date business intelligence sources.

### Parallel AI Search
Parallel AI is an advanced search and deep research API engineered specifically
to handle complex, multi-hop queries by synthesizing information across
scattered sources in a single pass.

### Tavily
Tavily is a search engine built from the ground up specifically for Large
Language Models (LLMs) and RAG workflows, returning synthesized answers rather
than just a list of blue links.
