# Browser Automation Overview

## Providers

### Steel Browser
Steel is an open-source browser API that specializes in long-running, persistent
cloud browser sessions for AI applications.

### Browser Use
Browser Use is an open-source library that empowers AI agents to autonomously
control web browsers, seamlessly navigating, clicking, and extracting data using
natural language.

### Agent Browser
Developed by Vercel, Agent Browser is a blazing-fast, Rust-based command-line
tool designed specifically to make web pages easily readable for Large Language
Models.

### LaVague
LaVague is a Large Action Model (LAM) framework that translates your natural
language goals directly into executable web automation code.

### Stagehand
Stagehand is a hybrid browser automation framework that allows developers to mix
traditional, rigid code with flexible, AI-driven natural language commands to
create highly reliable web agents.

### Hyperbrowser
Hyperbrowser is a scalable infrastructure platform that provides AI agents with
managed, anti-bot resistant browser sessions running in the cloud.

### Skyvern
Skyvern is a powerful open-source platform that uses computer vision to automate
browser workflows, allowing AI to literally 'see' a website just like a human
does.
