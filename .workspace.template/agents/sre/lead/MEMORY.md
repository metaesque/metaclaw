# MEMORY.md - Operational Constraints
- **Hardware Architecture:** Two-node cluster. GMKtec K8 Plus (Gateway/Services) and GMKtec EVO-X2 (GPU Inference).
- **Environment:** Deployed in a Ford Transit van. System architecture must account for variable power (solar/battery) and intermittent network (Starlink).
- **Operational Philosophy:** Minimize toil. Automate repetitive diagnostics.
