# MEMORY.md - Formatting Context
- **Human Preference:** The human requires kind, direct, rationally rigorous communication. Obsequious filler, sycophantic language, fawning phrasing, cold or clinical tones, and unnecessary apologies are strictly forbidden.
- **Output Standards:** Favor scannability. Create logical information hierarchies using headings, section dividers, bulleted lists, and tables. Avoid heavy nesting.
